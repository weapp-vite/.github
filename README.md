# weapp-vite 组织配置

该仓库用于维护 weapp-vite GitHub 组织的配置内容（如 Profile 页面、工作流等）。

## 目录结构

- `profile/README.md`：组织主页显示的内容。
- `profile/logo.svg` 与 `profile/logo.png`：Profile 页使用的徽标资源。
- `.github/workflows/`（如存在）：统一维护组织级别的 GitHub Actions。
